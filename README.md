## üöÄ Quick Start

Here's a simple example to get you started:

```python
import asyncio
from SafoneAPI import SafoneAPI

async def main():
    api = SafoneAPI()
    resp = await api.github("AsmSafone")
    print(resp.results)

asyncio.run(main())
```
